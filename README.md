### 開啟專案

---

1. Clone 此專案 `git clone https://github.com/viss5vuus98/Simple-Twitter-frontend`

---

### 專案環境建置

2. 使用終端機進入專案資料夾**Simple-Twitter-frontend**下輸入 `npm i` 引入相依套件

### 開啟專案

3. 在終端機專案資料夾路徑下輸入 `npm start`
4. 開始瀏覽器 輸入 URL ` http://localhost:3000/` 即可開啟專案
